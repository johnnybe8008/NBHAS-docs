Project Terms:

Term	                Use
Assessment	          The complete customer evaluation.
Assessment Session  	One completed assessment by a customer.
Assessment Report	    The generated PDF.
Assessment Summary	  The HTML email sent to the customer.
Assessment History	  All previous assessments for a customer.
Assessment Timeline	  Combined assessments, orders, and follow-up events.
Assessment Engine	    The software that drives the assessment.
Recommendation Engine	Determines products and instructions.
Progress Dashboard	  Shows changes across assessments.
Knowledge Library	    Your existing menopause articles, glossary, and educational resources.
